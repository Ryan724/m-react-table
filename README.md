m-react-table
============

m-react-table 

## Install & Run
    
    $ npm install r-scroller
    $ cd r-scroller
    $ npm install
    $ npm start

用浏览器打开`localhost:8080`,即可

## How to use 
  在你的代码当中如下使用
```javascript

    return (
      <div className ="table">
        <ReactTable 
          tableStyle = { tableStyle } 
          tableData = { tableData }>
        </ReactTable>
      </div>
      )

```
